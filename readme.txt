== Description ==

We all love chatting and use Telegram, Whatsapp, Facebook Messenger, Discord, Slack. Why not discuss contents of web-pages in the form of chatting? In modern days we can only use some commenting plugins. Discussion in a chat under post is innovation. Some says it is more convenient and more alive then comments.

ApeTail offers a fixed trigger button with unread messages notification which opens the widget frame on top of the page's content accessing to the Lobby chat/channel, direct chat with user (maybe with owner's of the web site), to AI Agent or to customly created chat/channel. Such button is located on any page).

#### â–¶ï¸ Under post
ApeTail offers messaging area under post; if not specified, chat name will be the same as title of the page. If you set rating:true , participants will be able to rate/like messages, so they appear sorted by rating.

#### â–¶ï¸ The Network
All ApeTail widgets are united to one network. If a user signed up to ApeTail in one web site, they will have their setting on other websites where ApeTail is installed. And, of course, can communicate to users directly who use ApeTail on other web sites and see followed/subscribed chats stream messages from other web sites.

== Installation ==

#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'ApeTail'
3. Activate ApeTail Communication System from your Plugins page.

#### Manually
Unzip to /wp-content/plugins directory, activate and change some settings if needed.
To avoid conflicts, deactivate other plugins that replace standard comments.
